##  Project Description
A fully responsive, professional website for "Refresh by Albader" - a fictional Bahraini business specializing in creative ice cream flavors and premium coffee. This project demonstrates web development skills using HTML5, CSS3, Bootstrap 4, and JavaScript.

##  Key Features
-  **Fully Responsive Design** - Works on mobile, tablet, and desktop
-  **Bootstrap 4 Framework** - Professional layout and components
-  **Image Carousel** - 3-slide product showcase
-  **Product Cards** - 3+ Bootstrap cards with hover effects
-  **Navigation Menu** - Collapsible for mobile devices
-  **Menu Gallery** - Grid layout of food items
-  **Professional Color Scheme** - Green theme matching business concept
-  **Interactive Elements** - Smooth scrolling, hover effects
-  **Bahraini Cultural Elements** - Integration of local flavors and themes

##  Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Custom styling and animations
- **Bootstrap 4.6** - Responsive framework
- **JavaScript/jQuery** - Interactive features
- **Font Awesome 5** - Icons
- **Google Fonts** - Typography

##  Responsive Testing
The website has been tested on:
- **Mobile:** iPhone SE, Samsung Galaxy (360px - 768px)
- **Tablet:** iPad (768px - 1024px)
- **Desktop:** Full HD monitors (1024px+)

##  Design Choices
### Color Scheme
- **Primary Green:** `#b0e0b0` - Represents freshness, nature
- **Dark Green:** `#226622` - For contrast and readability
- **Accent Yellow:** `#ffcc00` - Bahraini national color
- **Backgrounds:** Gradient of light greens for visual hierarchy

### Typography
- **Primary Font:** Arial/Helvetica (web-safe, professional)
- **Font Sizes:** Responsive using Bootstrap classes
- **Hierarchy:** Clear heading structure (h1-h6)
